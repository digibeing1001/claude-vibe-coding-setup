# VibeCoding Setup

完整说明请见 [README.md](README.md)。

这个仓库把一套可移植的 AI 编码工程配置打包给 Claude Code、Codex、Hermes、OpenClaw 等主流宿主使用。

核心能力包括：三层循环工程（L1 AI 编码 / L2 开发者反馈 / L3 用户反馈）、确定性控制器（8 状态转移）、STATE.md 状态脊柱、预算与 kill switch、Maker/Checker 物理分离、安全护栏（路径黑名单 / auto-merge / MCP 最小权限）、就绪度分层（L0/L1/L2/L3，18 维评分）、15 种失败模式目录、9 种循环模式、多循环协调、Post-Run Critique 自我修正、运行评分门禁、能力缺口发现、多宿主可移植、7 个 hook 节点自动化。

详细说明、安装命令、目录结构请直接看 [README.md](README.md)。
